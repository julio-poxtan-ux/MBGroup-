# Sección de Testimonios - Bootstrap 5.3

Proyecto de ejemplo que replica una sección de testimonios con dos filas
de tarjetas animadas con efecto de carousel infinito (marquee) en direcciones opuestas.

## Cómo usar

1. Abre `index.html` en tu navegador.
2. Reemplaza las imágenes de `assets/img/user-*.jpg` por las fotos reales de tus usuarios.
3. Ajusta los textos de los testimonios según tus necesidades.

La animación de movimiento se hace únicamente con CSS usando `@keyframes`.
